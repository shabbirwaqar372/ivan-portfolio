Vibes — Event Discovery App

UI/UX · Figma · Mobile prototype

Overview

Vibes is a mobile app concept that makes it easy to discover events in Stockholm. The goal was to create a friendly, lightweight experience where users can quickly see what’s happening near them without feeling overwhelmed.

What I did
  • Designed full UI for core flows and key screens.
  • Created low-fi wireframes and evolved them into high-fi Figma designs.
  • Built an interactive prototype with linked screens and user flows.
  • Collaborated with a team of 3 across research, ideation & visual design.
Highlights
  • Clear navigation with simple event categories and filters.
  • Soft, inviting visual language suited for social experiences.
  • Mobile-first layout with attention to spacing and readability.
  • Prototype ready for user testing and feedback sessions.

KickCapital — Rebranding Project

Branding · Visual identity · Figma

Overview

KickCapital is an investment company that needed a refreshed, coherent brand identity. As part of a Hyper Island client project, the goal was to create a flexible visual system that could live across web, decks and other communication channels.

What I did
  • Developed a complete brand book covering logo usage and lockups.
  • Defined typography scale, color systems & visual identity rules.
  • Contributed to layouts and structure for future website communication.
  • Translated client goals into a cohesive visual language in Figma.
Highlights
  • Clear brand system that can scale across digital and print.
  • Balanced professional tone with an approachable visual style.
  • Client-ready deliverables that support future web development.
